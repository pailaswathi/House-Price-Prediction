# House-Price-Prediction
House Price Prediction using Machine Learning with Kaggle’s House Prices dataset. Includes data cleaning, EDA, feature engineering, and regression model training to predict house sale prices.

Project Overview:

This project predicts house sale prices using machine learning based on various property features such as area, quality, rooms, and location. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, and training regression models.

Dataset:From Kaggle

Steps Performed:

Data cleaning (missing values handling, encoding categorical features)

Exploratory Data Analysis (EDA) and visualization

Feature engineering and scaling

Model training using regression algorithms

Model evaluation and prediction generation

Technologies Used

Python

Pandas, NumPy

Matplotlib, Seaborn

Scikit-learn

Output

The model generates predicted house prices for the test dataset and creates a submission file for Kaggle.
